# ATV/UTV Reference Dashboard

## Overview
B2B ATV/UTV 납품 데이터를 기반으로 구축한 레퍼런스 분석 대시보드입니다.

## Features
- 산업군별 납품 분석
- 모델별 사용 현황 분석
- 고객 레퍼런스 검색
- 이미지 기반 사례 확인
- 출처 링크 연결

## Tech Stack
- Streamlit
- Pandas
- Plotly
- Parquet

## Deployment
Streamlit Community Cloud (Git 기반 자동 배포)

## Business Value
- 영업 레퍼런스 즉시 탐색
- 산업별 타겟 전략 수립
- 제품 포트폴리오 분석
